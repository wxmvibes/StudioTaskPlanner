# StudioTaskPlanner

**A lightweight, browser‑based task‑planning tool designed specifically for Roblox Studio developers. This website helps creators organize their workflow, track progress, and manage multi‑step tasks while building games, systems, and experiences in Roblox Studio.**

## 🌟 Features

- **Task Creation & Management**  
  **Add, edit, and remove tasks with ease. Perfect for scripting, building, UI work, bug‑tracking, and more.**

- **Progress Tracking**  
  **Mark tasks as complete and visually track your development progress.**

- **Clean, Developer‑Friendly UI**  
  **Simple layout designed to stay out of your way while you work.**

- **Local Storage Support**  
  **Your tasks persist between sessions—no login required.**

- **Roblox‑Focused Workflow**  
  **Built around the typical development cycle of Roblox Studio projects.**

## 📦 Installation & Setup

**You can run the site locally in just a few steps:**

**1. Clone the repository:
   git clone https://github.com/your-username/your-repo-name.git
2. Open the project folder.
3. Launch `index.html` in your browser.**

**No build tools or dependencies required.**

## 🧭 Usage

**1. Open the website.
2. Add tasks related to your Roblox Studio project.
3. Mark tasks as complete as you progress.
4. Refresh or close the tab—your tasks stay saved automatically.**
